Projects

Vehicle Speed Detector

Designed a system using Arduino and two IR sensors placed at a fixed distance. The Arduino measures the time taken for a vehicle to pass between the sensors and calculates its speed, displaying the result on an LCD screen.

Smart Blind Stick

Developed a smart walking stick using Arduino, integrated with an ultrasonic sensor for obstacle detection and a color sensor to identify surface colors. The system alerts the user through vibrations or sound based on proximity and color feedback, enhancing navigation assistance for visually impaired individuals.

Credit Card Fraud Detection

Implemented a machine learning model using the Decision Tree algorithm to detect fraudulent credit card transactions. The model was trained on transaction data to classify activities as genuine or fraudulent based on patterns in features like transaction amount, time, and location.
